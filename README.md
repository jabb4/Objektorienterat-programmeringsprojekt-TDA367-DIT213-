# Roguelike Game

> Project for TDA367/DIT213

A game built in Java using JavaFX.

---

## Built with

- Java 25
- JavaFX 25.0.1
- Maven

---

## Installation

1. Clone the repository and cd into it
```bash
git clone https://github.com/jabb4/Objektorienterat-programmeringsprojekt-TDA367-DIT213-.git
cd "Roguelike Game"
```

2. Run the game
```bash
mvn javafx:run
```
